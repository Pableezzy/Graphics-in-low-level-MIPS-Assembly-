Lab 5: Function and Graphics

----------
DESCRIPTION

This program will perform some primitive graphics operations 
on a simulated Bitmap display. The functions utilized will  
fill the display with a given color, create  
rectangular and diamond shapes using a memory - mapped
graphic display tool in MARS called Bitmap. 

----------
Files

-
Lab5.asm

This file includes the assembly code of the lab.

----------
INSTRUCTIONS

This program is intended to be run using the MIPs Assembler and Runtime Simulator (MARS).
To visualize the graphics, access the Bitmap display tool by selecting Tools->Bitmap Display
in the MARS simulator.
